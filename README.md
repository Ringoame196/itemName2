# itenName2

## プラグイン説明
手持ちのアイテムの名前、説明欄、カスタムモデルデータを1コマンドで変更することが可能 <br>
&で色つけ可能に

## コマンド
/itemname [種類] [入力]

## 使い方
変更したいアイテムを手持ちに持ち /itemname [種類] [入力]で変更

## 開発環境
- Minecraft Version : 1.20.1
- Kotlin Version : 1.6.10
